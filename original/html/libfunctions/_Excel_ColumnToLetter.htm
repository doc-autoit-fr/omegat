Function Reference

_Excel_ColumnToLetter

Converts the column number to letter(s)

#include <Excel.au3>
_Excel_ColumnToLetter ( $iColumn )

Parameters

$iColumn The column number which you want to turn into letter(s)

Return Value

Success: the column letter(s).
Failure: "" and sets @error.
@error: 1 - $iColumn is not a number

Related

_Excel_ColumnToNumber

Example

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; ****************************************************************************
; Translate an Excel column letter to the colunm number
; ****************************************************************************
Local $sCol = "YZ"
Local $iNumber = _Excel_ColumnToNumber($sCol)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_ColumnToNumber Example 1", "Error converting letter to number." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_ColumnToNumber Example 1", "Letter: " & $sCol & " = Number: " & $iNumber)
