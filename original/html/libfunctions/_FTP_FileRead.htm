Function Reference

_FTP_FileRead

Reads data from a handle opened by _FTP_FileOpen()

#include <FTPEx.au3>
_FTP_FileRead ( $hFTPFile, $iNumberOfBytesToRead )

Parameters

$hFTPFile Handle returned by _FTP_FileOpen() to the ftp file.
$iNumberOfBytesToRead Number of bytes to read.

Return Value

Success: the binary/string read.
Failure: 0 and sets the @error flag to non-zero.
@error: -1 for end-of-file
Non-zero for other errors.
