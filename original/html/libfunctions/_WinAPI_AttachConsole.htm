Function Reference

_WinAPI_AttachConsole

Attaches the calling process to the console of the specified process

#include <WinAPI.au3>
_WinAPI_AttachConsole ( [$iPID = -1] )

Parameters

$iPID [optional] Identifier of the process. Set to -1 to attach to the current process.

Return Value

Success: True
Failure: False
