_WinAPI_SetCursor

Establishes the cursor shape

#include <WinAPI.au3>
_WinAPI_SetCursor ( $hCursor )

Parameters

$hCursor Identifies the cursor

Return Value

Success: The handle of the previous cursor, if there was one.
Failure: 0, if there was no previous cursor.
