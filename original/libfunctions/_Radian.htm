Function Reference

_Radian

Converts degrees to radians

#include <Math.au3>
_Radian ( $iDegrees )

Parameters

$iDegrees Degrees to be converted into radians.

Return Value

Success: the radians converted from degrees.
Failure: an empty string and sets the @error flag to non-zero.

Remarks

Can be used in conjunction with the Sin(), Cos(), and Tan() functions.

Related

_Degree

Example

#include <Math.au3>
#include <MsgBoxConstants.au3>

Local $fDegrees = 35
Local $fRadians = _Radian($fDegrees)

MsgBox($MB_SYSTEMMODAL, "Degrees to Radians", $fDegrees & " degrees = " & $fRadians & " radians")
